      브라우저 단축키
      

      

      Ctrl + n
      새 창 열기
      

      

      Ctrl + w
      현재 탭 닫기
    

    

      Ctrl + t
      새 탭을 열어 이동
    

    

      Ctrl + Shift + t
      마지막으로 닫은 탭 다시 열어 이동
    

    

      Ctrl + Tab or Ctrl + PgDn
      다음에 연 탭으로 이동
    

    

      Ctrl + Shift + Tab or Ctrl + PgUp
      이전에 연 탭으로 이동
    

    

      Ctrl + 1~8
      특정 탭으로 이동
    

    

      Ctrl + 9
      마지막 탭으로 이동
